# Worked examples

> **Status:** Planned. This directory will collect end-to-end worked examples — taking a model from raw geometry through to a working DCS export.

## Examples planned

- **Ship export** — full bridge-to-keel build using the LOD whole method
- **Aircraft export** — bone-driven control surfaces, animated gear, weapon connectors
- **Ground vehicle export** — wheel rotation, turret traverse, hull number decal

Each example will be a single markdown file walking through the complete process, with notes on common pitfalls at each step.

## Want to contribute an example?

Examples are the most accessible way for new modders to learn the pipeline. If you have a model you've taken end-to-end and would be willing to write up the process, please open a [content addition issue](../.github/ISSUE_TEMPLATE/content_addition.md).
